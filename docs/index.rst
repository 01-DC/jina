Welcome to Jina
******************

Jina is a deep learning-powered search framework for building cross-/multi-modal search systems (e.g. text, images, video, audio) on the cloud.

Quick links
===========

- Read up on `neural search <./get_started/neural-search>`_ and `Jina's key concepts <./get_started/concepts>`_
- `Install Jina <./get_started/install>`_
- `Build your first Jina app <./tutorials/chatbot>`_
- `Troubleshoot your problems <./support/troubleshooting>`_
- `Talk to us on Slack <https://slack.jina.ai>`_

How to use our docs
===================

We've broken our docs down into different sections to help guide you on your Jina journey:

- **Get started**: Key concepts, how to install Jina, and how to run our hello world apps
- **Build with Jina**: Step-by-step guides and starter kits to build and extend your Jina app
- **Understand Jina**: Dive deeper with our cookbooks and learn how to make the most of Documents, Executors, Flows, and more
- **Developer reference**: How our API and CLI work
- **Support**: Troubleshooting and how to get help
